== Description ==

Widget that Get all facebook page feed with cover image and count likes

This Widget is created for facebook page feed.

== Features ==

1. Show facebook page Friends Faces.
2. Use facebook page Small Header.
3. Hide facebook page Cover Photo.
4. Show all facebook page feed.
5. It will show the current activity of your page.


== Installation ==
1. Unzip and upload the `my5 fb page feed` directory to the plugin directory ('/wp-content/plugins/')
2. Activate the plugin through the 'Plugins' menu in WordPress
